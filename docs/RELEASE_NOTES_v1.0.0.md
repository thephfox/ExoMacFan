## Highlights

- Public release of ExoMacFan with universal macOS binaries (`arm64` + `x86_64`)
- Pro-Active fan mode with dynamic anti-throttling behavior
- Improved fan mode persistence and active mode visibility in the Fans tab
- Updated documentation for architecture, thermal sensors, and distribution workflow
- License includes attribution requirement and explicit hardware-risk disclaimer

- `ExoMacFan.app` (universal binary)

## Notes

- Fan control requires admin privileges for helper daemon installation.
- Monitoring-only features work without elevated permissions.
- Supported platforms: Apple Silicon (M1–M5) and Intel Macs.
